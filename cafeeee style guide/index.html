INTERACTIVE ELEMENTS

Navigation Button

CSS selector:

.header-nav-size

Sample code:

<a href="#" class="button-nav">Default Button</a>

Rendered element:

Home

In-text link

CSS selector:

.inline-text

Sample code:

<a href="#" class="inline-text">inline-text</a>

Rendered element:

Browse

Submit/Reset Buttons

CSS selector:

.buttons

Sample code:

<button type="submit" class="confirm-nav">Confirm</a>

Rendered element:

Confirm

Text Input Field

</form>

.list.header-list

<h2>Header Level 1</h2>
</li>
</ul>

</li>
</ul>
